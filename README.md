## 🚀 Project Overview

This project is an interactive **Data Visualization Dashboard** built using Python and Dash.
It allows users to explore FIFA World Cup match data dynamically using multiple visualization techniques.

The dashboard provides insights into match statistics such as goals, trends over the years, and overall match performance.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Dash

---

## 📈 Features

* 📊 Multiple Graph Types (Bar, Line, Scatter, Histogram)
* 🎯 Dynamic Attribute Selection
* ⚡ Interactive Dashboard UI
* 📅 Year-wise Data Visualization
* 📉 Data Cleaning and Processing

---

## 📂 Dataset

The dataset used in this project contains historical FIFA World Cup match data.

👉 Source: Kaggle FIFA World Cup Dataset

## ▶️ How to Run the Project

### Step 1: Clone the repository

git clone https://github.com/your-username/fifa-worldcup-dashboard.git

### Step 2: Navigate to project folder

cd fifa-worldcup-dashboard

### Step 3: Install dependencies

pip install -r requirements.txt

### Step 4: Run the application

python app.py

### Step 5: Open in browser

http://127.0.0.1:8050/
